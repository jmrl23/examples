# image compression

basic web server that serve static images into its compressed version (if defined).

## Quick start

1. install dependencies

   ```bash
   yarn install
   ```

1. run build script (optional for development mode)

   ```bash
   yarn run build
   ```

1. run start script

   ```bash
   # development
   yarn run start:dev
   ```

   ```bash
   # built
   yarn run start
   ```
