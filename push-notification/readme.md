# push notification

webhook push notification

#### Note

make sure the you grant the notification permission on browser

## Quick start

1. install dependencies

   ```bash
   yarn install
   ```

1. run vapid generation

   ```bash
   yarn run gen:vapid
   ```

1. run build script (optional for development mode)

   ```bash
   yarn run build
   ```

1. run start script

   ```bash
   # development
   yarn run start:dev
   ```

   ```bash
   # built
   yarn run start
   ```
